# 🛠️ ServiHub - Local Service Finder

**ServiHub** is a web application that helps users book **local service providers** (like electricians, plumbers, tutors, etc.) quickly and conveniently. It provides easy scheduling, service tracking, and user reviews, making it simple to connect with trusted professionals in your area.

---

## 🚀 Features

* 🔍 Search and browse local service providers
* 📅 Book appointments with time-slot selection
* ⭐ Rate & review service providers
* 🔔 Real-time booking status updates
* 👤 Separate dashboards for customers and service providers

---

## 🏗️ Tech Stack

**Frontend**

* React.js / Vite
* Tailwind CSS
* Redux Toolkit (for state management)

**Backend**

* Node.js + Express.js
* MongoDB (Database)
* JWT Authentication

**Deployment**

* Frontend → [Netlify](https://www.netlify.com/)
* Backend → [Render](https://render.com/)

## ⚡ Getting Started

### Clone Repository

```bash
git clone https://github.com/your-username/servihub.git
cd ServiHub
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```
